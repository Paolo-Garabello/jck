# JCK
The Java-Bases Chat Kit.

## Requirements
* Docker
* Docker-compose
* Make (optional)

## Installation
1. Clone this repo:
   ```bash
   git clone https://github.com/Paolo-Garabello/jck.git
   ```
2. Start the app:
   * With make:
     ```bash
     make
     ```
   * With docker-compose:
     ```bash
     docker-compose build
     docker-compose up
     ```
3. Open the frontend at `localhost:3000` (*or any other available network interface*)
